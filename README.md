## 📖 About The Project  
💡 This is a **Full-Stack E-Commerce Web Application** built using modern web technologies.  
It uses **Server-Side Rendering (SSR)** with EJS to deliver **fast, dynamic, and SEO-friendly pages**.

✨ Users can explore products, view detailed information, and manage inventory efficiently.

---

## 🚀 Features  

✨ **Core Functionalities**
- 🖥️ Server-Side Rendering (SSR) using EJS  
- 🛍️ Product listing and detailed pages  
- ➕ Add, ✏️ Edit, ❌ Delete products (CRUD)  
- 🗄️ MongoDB database integration  
- 📱 Fully responsive UI with Bootstrap  
- 🔗 RESTful APIs with Express.js
